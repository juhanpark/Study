# 언리얼 개념  

Component = 객채  

Actor  
- 레벨에 배치할 수 있는 오브젝트
- 실체를 만들때 상속받음
- Component 사용간으
- 좌표 실체화할때 사용하지않고 이때 SceneComponent사용


Super  
- 부모에 있는 virtual을 추가 실행  

Asset 에셋  
- 게임을 만드는데 사용되는 요소  
-콘텐츠 브라우저에서 임포트하거나 만들 수 있는 콘텐츠, 일례로 머티리얼, 스태틱 메시, 텍스처, 파티클 시스템, 블루프린트, 사운드 큐 등

Mapping
일치화
